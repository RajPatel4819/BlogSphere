# BlogSphere - Online Blog Management System

BlogSphere is a lightweight, responsive Mini Project developed for the Web Programming (3160713) course. It allows users to create, manage, and share blog posts, while providing an admin interface for system moderation.

## Features

- **User Authentication**: Secure registration, login, and logout.
- **Role-based Access**: Separate interfaces for Users and Administrators.
- **Blog Management**: Full CRUD (Create, Read, Update, Delete) functionality for blog posts.
- **Categorization**: Organize blogs by categories like Technology, Travel, and Education.
- **Search & Filter**: Easily find blogs based on titles or categories.
- **Premium Design**: Built with Bootstrap 5 for a modern, responsive user experience.

## Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript (jQuery, AJAX)
- **Framework**: Bootstrap 5
- **Backend**: PHP
- **Database**: MySQL

## Setup Instructions

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/RajPatel4819/BlogSphere.git
    ```
2.  **Move to XAMPP**:
    Place the `BlogSphere` folder into your `xampp/htdocs/` directory.
3.  **Database Configuration**:
    - Open PHPMyAdmin.
    - Create a database named `blog_sphere_db`.
    - Import the `db_schema.sql` file.
4.  **Access the Application**:
    Navigate to `http://localhost/BlogSphere/` in your browser.
